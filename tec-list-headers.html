NAME

tec-list-headers — temporary email commands: list headers of messages

SYNOPSIS

tec-list-headers -k | -s

DESCRIPTION

Input
tec-list-headers requires the information previously created by tec-create-address(1) to be on its standard input.

Arguments and Options
tec-list-headers does not take any arguments.

-k

Use burner.kiwi temporary email service.

-s

Use 1secmail temporary email service.

EXAMPLES

burner.kiwi

tec-list-headers -k < kiwi-account.json

1secmail

tec-list-headers -s < secmail-address.txt
